# Code of Conduct

Be civil. Disagree on technical substance, not on people. Assume good faith; correct it when proven otherwise.

The project follows the standard expected of any first-party AGNOS repo: contributors and users alike are welcome regardless of background. Harassment, personal attacks, doxxing, or bad-faith engagement get warned once and then banned.

Report concerns privately — same channel as security reports in [`SECURITY.md`](SECURITY.md). Disputes that affect contribution review are handled by the maintainer; appeals are heard but final.

This code applies to all project spaces: issues, PRs, code review, and any chat or forum the project formally adopts.
